# Explainable Gene Influence Analyzer (E-GIA)

## Overview
This project analyzes gene expression differences between healthy and disease conditions and provides interpretable insights into gene influence.

## Features
- Computes gene-wise expression differences
- Identifies disease-associated and healthy-associated genes
- Generates human-readable explanations

## Why this project
To simulate how computational analysis can help interpret biological patterns and identify potential biomarkers.

## Future Scope
- Apply to real gene expression datasets
- Integrate statistical testing
- Extend to multi-gene interaction analysis
